# Tic Tac Toe DApp

## Requirements

This project is built with:

- [React](https://reactjs.org/)
- [Web3.js](https://web3js.readthedocs.io/en/1.0/index.html)

## Setup

```bash
# install dependencies
npm install

# serve at localhost:3000 
npm start

# run tests
npm test

# build for production
npm run build
```
